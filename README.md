# Unemployment Analysis with Python

## 📌 Project Overview

This project analyzes unemployment trends in India using Python. It includes data cleaning, exploratory data analysis (EDA), visualization, and an analysis of the impact of COVID-19 on unemployment rates.

This project was completed as part of the **CodeAlpha Data Science Internship**.

---

##  Objective

The objective of this project is to:

- Analyze unemployment rates across different regions of India.
- Compare unemployment in Urban and Rural areas.
- Study monthly unemployment trends.
- Understand the impact of COVID-19 on unemployment.
- Generate insights using data visualization.

---

## 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
- Google Colab

---

## 📂 Dataset

This project uses two datasets:

1. **Unemployment in India.csv**
2. **Unemployment_Rate_upto_11_2020.csv**

The datasets contain:

- Region
- Date
- Area (Urban/Rural)
- Estimated Unemployment Rate (%)
- Estimated Employed
- Estimated Labour Participation Rate (%)

---

## 📊 Project Workflow

1. Import Libraries
2. Load Dataset
3. Clean Data
4. Handle Missing Values
5. Exploratory Data Analysis
6. Data Visualization
7. COVID-19 Impact Analysis
8. Generate Insights

---

## 📈 Visualizations

The project includes the following visualizations:

- Average Unemployment Rate by Region
- Urban vs Rural Unemployment
- Monthly Unemployment Trend
- COVID-19 Unemployment by Region
- Top 10 Regions with Highest Unemployment

---

## 🔍 Key Insights

- Unemployment rates differ significantly across regions.
- Urban and Rural areas show different unemployment patterns.
- COVID-19 increased unemployment in several regions.
- Monthly unemployment trends fluctuate over time.
